Abstract


   CSPICE_UCRSS compute the normalized cross product of two
   3-vectors.

   For important details concerning this module's function, please refer to
   the CSPICE routine ucrss_c.

I/O


   Given:

      vec1,
      vec2   two arbitrary double precision 3x3 matrices.

   the call:

      cspice_ucrss, vec1, vec2, vout

   returns

      vout   a double precision 3x3 matrix, the normalized cross product
             of 'vec1' and 'vec2'.

Examples


   Native IDL code to perform the same operation:

      vout = crossp( vec1, vec2) / norm( crossp(vec1,vec2) )

   The IDL functions accept arbitrary size N-vectors.
