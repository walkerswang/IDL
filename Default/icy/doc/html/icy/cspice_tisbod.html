Abstract


   CSPICE_TISBOD calculates the 6x6 double precision matrix that 
   transforms state vectors in inertial coordinates to state
   vectors in body-equator-and-prime-meridian coordinates.

   For important details concerning this module's function, please refer to
   the CSPICE routine tisbod_c.

I/O

   
   Given:
   
      ref    the string ID of an inertial reference frame 
      body   the integer NAIF ID of a body of interest
      et     the double precision ephemeris time of interest
   
   the call:
   
      cspice_tisbod, ref, body, et, tsipm
   
   returns:
   
      tsipm   a double precision 6x6 transformation matrix between
              'ref' and the body-equator-and-prime-meridian frame 
              of 'body'
   
   Note: NAIF recommends use of cspice_spkezr with the appropriate 
   frames kernels when possible over cspice_tisbod.

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.

   
      ;;
      ;; Calculate the matrix to rotate a state vector from the
      ;; J2000 frame to the Saturn fixed frame at a specified
      ;; time. First load the needed kernels.
      ;;
      cspice_furnsh, 'standard.tm'
   
      ;;
      ;; The body ID for Saturn
      ;; 
      sat_ID = 699
   
      ;;
      ;; Retrieve the transformation matrix at some time.
      ;;
      cspice_str2et, 'Jan 1 2005', et
      cspice_tisbod, 'J2000', sat_ID, et, tsipm
   
      ;;
      ;; Rotate a state 6-vector (state_vec) into the 
      ;; Saturn fixed frame.
      ;;
      sat_vec = tsipm ## state_vec
