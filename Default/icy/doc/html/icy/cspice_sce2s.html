Abstract


   CSPICE_SCE2S converts an ephemeris seconds past J2000 (ET) 
   value describing a date to a character string representation 
   of a spacecraft clock value (SCLK). 

   For important details concerning this module's function, please refer to
   the CSPICE routine sce2s_c.

I/O

   
   Given:
   
      sc   the scalar integer NAIF ID of the spacecraft whose 
           encoded SCLK value at the epoch 'et' is desired

      et   the scalar double precision epoch, specified as ephemeris 
           seconds past J2000 TDB
   
   the call:
   
      cspice_sce2s, sc, et, sclkch
   
   returns:
   
      sclkch   the scalar character string representation of the
               spacecraft clock value that corresponds to 'et', for
               the spacecraft 'sc'

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.
   
      ;;
      ;; Load the leapseconds kernel for time conversion.
      ;;
      cspice_furnsh, 'standard.tm'
   
      ;;
      ;; Assign values for the spacecraft ID (Voyager2),
      ;; s-clock kernel.
      ;;
      SC         = -32
      SCLK       = '/kernels/voyager2/sclk/vg200004.tsc'
      event_time = '1979 JUL 05 21:50:21.23379'
   
      ;;
      ;; Load the SCLK kernel.
      ;;
      cspice_furnsh, SCLK
   
      ;;
      ;; Convert the time string to ephemeris time.
      ;;
      cspice_str2et, event_time, et
   
      ;;
      ;; Convert the ephemeris time to the corresponding
      ;; s-clock string appropriate for this spacecraft
      ;;
      cspice_sce2s, sc, et, sclkch
      print, sclkch

      ;;
      ;; It's always good form to unload kernels after use,
      ;; particularly in IDL due to data persistence.
      ;;
      cspice_unload, SCLK
      cspice_unload, 'standard.tm'

   IDL outputs: 2/20538:39:768
