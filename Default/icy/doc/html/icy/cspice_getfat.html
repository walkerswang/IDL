Abstract


   CSPICE_GETFAT determine the file architecture and file type of
   most SPICE kernel files.

   For important details concerning this module's function, please refer to
   the CSPICE routine getfat_c.

I/O

   
   Given:
   
      file    the string name of a SPICE kernel file
   
   the call:
   
      cspice_getfat, file, arch, type
   
   returns:
   
      arch   a string describing the file's architecture
      type   a string describing the file type

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.

   
      cspice_getfat, '/kernels/gen/lsk/naif0008.tls', arch, type
      print, arch + ' ' + type
   
   IDL outputs:   KPL LSK
   A kernel pool file of LSK (leapseconds) type.
   
      cspice_getfat, '/kernels/gen/spk/de405_2000-2050.bsp', arch, type
      print, arch + ' ' + type
   
   IDL outputs:   DAF SPK
   A DAF file of SPK type.
   
      cspice_getfat, '/kernels/gen/pck/pck00008.tpc', arch, type       
      print, arch + ' ' + type
   
   IDL outputs:   KPL PCK
   A kernel pool file of PCK type.
