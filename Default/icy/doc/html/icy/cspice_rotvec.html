Abstract


   CSPICE_ROTVEC transform a 3-vector to a new coordinate
   system, the system generated by the rotation of angle 
   radians about axis. This transformation rotates 
   v1 by -angle (minus angle) radians about the specified 
   axis.

   For important details concerning this module's function, please refer to
   the CSPICE routine rotvec_c.

I/O

   
   Given a double precision 3-vector, a double precision
   scalar angle of rotation 'angle' measured in radians, 
   and the integer index 'iaxis', identifying the axis of 
   rotation (1, 2, or 3 where X=1, Y=2, Z=3), the call:
   
      cspice_rotvec, v1, angle, iaxis, vout
   
   returns to 'vout' a double precision 3-vector
   
   The call has the effect of:
   
      cspice_rotate, angle, iaxis, rot_mat
      cspice_mxv, rot_mat, v1, vout
