Abstract


   CSPICE_KTOTAL returns the current number of kernels of
   a specific type loaded via cspice_furnsh.

   For important details concerning this module's function, please refer to
   the CSPICE routine ktotal_c.

I/O

   
   Given:
   
      kind    the string identifier indicating the type of kernels
              to count
   
   the call:
   
      cspice_ktotal, kind, count
   
   returns:
   
      count   the integer number of 'kind' type files loaded by
              the keeper subsystem
