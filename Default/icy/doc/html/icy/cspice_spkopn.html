Abstract


   CSPICE_SPKOPN creates then opens a new SPK file, returning
   the handle of the opened file.

   For important details concerning this module's function, please refer to
   the CSPICE routine spkopn_c.

I/O

   
   Given:
   
      fname    a scalar string naming the CK file to open
               
      ifname   a scalar string naming the descriptive internal 
               filename for the CK

      ncomch   the scalar integer number of characters to 
               reserve for comments. 
   
   the call:
   
      cspice_spkopn, name, ifname, ncomch, handle
   
   creates, opens, and prepares for writing the SPK file 
   'name' returning a scalar integer, 'handle', that refers
   to 'name'.

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.

   
      ;;
      ;; Define the SPK file parameters.
      ;;
      SPK    = 'new_spk.bsp'
      ifname = 'Type 2 SPK internal file name.'
      ncomch = 10
   
      ;;
      ;; create/open the file.
      ;;
      cspice_spkopn, SPK, ifname, ncomch, handle
   
         ... write to the file  ...
      ;;
      ;; SAFELY close the SPK file.
      ;;
      cspice_spkcls, handle
