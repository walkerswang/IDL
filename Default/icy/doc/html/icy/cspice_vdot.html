Abstract


   CSPICE_VDOT returns a scalar double describing the dot product
   (inner product) of two double precision 3-vectors. An error 
   signals if the inputs are not 3-vectors.

   For important details concerning this module's function, please refer to
   the CSPICE routine vdot_c.

I/O

   
   Given two arbitrary double precision 3-vector 'vec1' 
   and 'vec2': 
   
     double_dot = cspice_vdot (vec1,vec2)
                                                  
   returns the dot product ( <vec1,vec2> ) to the scalar 
   variable 'double_dot'.

Examples


   Native IDL code to calculate the same scalar result:
   
      double_dot = transpose(vec1) # vec2
      double_dot = vec2 ## transpose(vec1) 
   
   The native IDL function accepts arbitrary sized N vectors.
