Abstract


   CSPICE_VSEP returns the separation angle between two 3-vectors.

   For important details concerning this module's function, please refer to
   the CSPICE routine vsep_c.

I/O


   Given:

      v1   is an arbitrary double precision, 3-vector.

      v2   is also an arbitrary double precision, 3-vector
           vector.

           'v1' or 'v2' or both may be the zero vector.

      An implicit assumption exists that 'v1' and 'v2' lie in the same
      reference frame. If this is not the case, the numerical result has
      no meaning.

  the call:

      vsep = cspice_vsep( v1, v2 )

   returns:

      vsep   the double precision, positive definite, scalar angular separation
             between 'v1' and 'v2' expressed in radians. If either 'v1' or 'v2'
             is the zero vector, the return value has value zero.
