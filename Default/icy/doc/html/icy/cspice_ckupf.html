Abstract


   CSPICE_CKUPF unloads a CK pointing file from the kernel
   system.

   For important details concerning this module's function, please refer to
   the CSPICE routine ckupf_c.

I/O

   
   Given:
   
      handle   the scalar integer handle to a loaded CK file
   
   the call:
   
      cspice_ckupf, handle
   
   removes the file from the kernel system. Data searches will not
   consult the data contained in the unloaded file.
   
   Note: NAIF recommends use of cspice_furnsh to load and
   cspice_unload to respectively load then unload CK kernels;
   cspice_ckupf is a deprecated routine.

Examples


   Any numerical results shown for this example may differ between
   platforms as the results depend on the SPICE kernels used as input
   and the machine specific arithmetic implementation.

   
      ;;
      ;; Given the name of some CK file CK_FILE, load 
      ;; it for data access.
      ;;
      cspice_cklpf, CK_FILE, handle
   
         ... perform calculations, but do not change handle ...
   
      ;;
      ;; Unload the CK file. Uses cspice_ckupf only in those
      ;; situations where you loaded the CK using cspice_cklpf.
      ;;
      cspice_ckupf, handle
