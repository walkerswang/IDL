Abstract


   CSPICE_VUPACK unpacks three scalar double precisions components
   from a row form 3-vector.

   For important details concerning this module's function, please refer to
   the CSPICE routine vupack_c.

I/O


   Given:

      vec   an arbitrary double precision 3-vector with components x, y, z:

            vec = [ x, y, z ]

   the call:

      cspice_vupack, vec_in, x, y, x

   returns:

      x,
      y,
      z   the double precision scalar components of 'vec'.

   The action of cspice_vupack reverses the affect of cspice_vpack.

Examples


   The native IDL code to perform this operation:

      x = vec[0]
      y = vec[1]
      z = vec[2]
